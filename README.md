## Features
- VLESS, VMess, Trojan support via Xray-core
- Import from clipboard, deep link and QR code
- Subscription updates with traffic/expiry info
- Server list with country flags and ping check
- Light/Dark/System theme
- Windows modes: `system_proxy` / `tunnel`

## Build Windows Installer
1. Install [Inno Setup 6](https://jrsoftware.org/isdl.php)
2. Run:

```powershell
cd C:\Users\User\Desktop\chrnet
powershell -ExecutionPolicy Bypass -File .\scripts\build-installer.ps1
```

Output: `dist\ChrNet-Setup-<version>.exe`
